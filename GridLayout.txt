In javaFX, the grid layout is described as such:

GridPane grid = new GridPane(); //generate the GridPane overlay object

grid.add(Object, x,y,z,a);

//Where Object = object to be added to grid
//Where x = horizontal offset
//Where y = verticle offset
//Where z = row offset
//Where z = col offset
/*
Described within https://docs.oracle.com/javafx/2/api/javafx/scene/layout/GridPane.html as:

add(Node child, int columnIndex, int rowIndex, int colspan, int rowspan)
Adds a child to the gridpane at the specified column,row position and spans.

*/
